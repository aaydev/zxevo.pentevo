# Версии плат ZX Evolution от Олега Solegstar

Версии плат идут в следующем порядке (сперва старые, затем новые):
- **01_rev_b3** : Revision B3
- **02_rev_cs** : Revision CS
- **03_rev_cv** : Revision CV
- **04_rev_cs2** : Revision CS2

Задние панели для различных версий ZX Evolution в каталоге **backplane_zxevo**.

## Перечень изменений

### Revision B3:
Изменения от версии B (B от NedoPC):
 - Добавлено управление множителем частоты;
 - Используется ПЗУ в DIP корпусе;
 - Используется ОЗУ в TSOP II - 50 корпусе;
 - Заменен PAL-кодер на AD724;
 - Добавлены футпринты под разные держатели SD-карт: полноразмерный, укороченный и micro-SD.
 
### Revision CS:
Изменения:
- Added HDAUDIO connector.
- Added MICRO-SD slot.
- X9 is KEMPSTON SEGA GAMEPAD connector, not RS232 now.
- X19 is RS232 TTL-standard.
- BAT54C replaced by SS14.
- ATX connector is universal for 4.2/5.5 pins step.

### Revision CV:
Изменения:
- Added HDAUDIO connector.
- Added MICRO-SD slot.
- X9 is KEMPSTON SEGA GAMEPAD connector, not RS232 now.
- X19 is RS232 TTL-standard.
- BAT54C replaced by SS14.
- ATX connector is universal for 4.2/5.5 pins step.
- Added EXT_VGA connector.

### Revision CS2:
Изменения:
- Added HDAUDIO connector.
- Added MICRO-SD slot.
- X9 is KEMPSTON SEGA GAMEPAD connector, and second redefine GAMEPAD, not RS232 now.
- X19 is RS232 TTL-standard.
- BAT54C replaced by SS14.
- ATX connector is universal for 4.2/5.5 pins step.
- Added EXT_VGA connector for External video source, like VDAC2 or etc.
- Replaced FT232 to CP2102.
